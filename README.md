# Get a quick weather forecast from 'Classy Weather' for any location!

## Get to know-

- Weather condition
- Highest and lowest temperatures
- <em> A 7-day forecast </em>

---

## Installation

Install "Classy Weather" with npm

```bash
  npm install
  npm run start
```
